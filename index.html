Dependencies

There are no dependencies to install in order to use the plugin.

If courgette assets are missing - it should not happen as install script takes care of it. However, if something went wrong, you could do a manual setup by creating a symlink to courgette/public/courgette inside your application's public directory : $ cd public $ ln -s ../vendor/plugins/courgette/public/courgette courgette To contribute: * fork the project * bundle install

Install

script/plugin install git@github.com:21croissants/courgette.git
And browse to:

/features Voilà, pretty list of features available in all env except "production". Great for communicating with the customer.

Contribute

You can clone the project with Git by running: 

git clone git://github.com/21croissants/courgette
