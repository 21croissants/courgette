courgette by 21croissants

Courgette is a Rails engine (plugin) for Rails 2.x applications which makes your cucumber features files viewable through your browser

Install

script/plugin install git@github.com:21croissants/courgette.git
And browse to:

/features Voilà, pretty list of features available in all env except for "production". Great for communicating with the customer.

Dependencies

There are no dependencies to install in order to use the plugin. 

This plugin has been tested with ruby 1.8.7 and Rails 2.3.8

install.rb script takes care of adding a symlink to assets. However, if something went wrong, you could do a manual setup by creating a symlink to courgette/public/courgette inside your application's public directory :

  cd public
  ln -s ../vendor/plugins/courgette/public/courgette courgette

Contribute

You can contribute to the project with Git:

  • Fork the project.
bundle install
rake # should have no failures
  • Make your feature addition or bug fix (pls specify ruby & rails versions)
  • Add tests for it. This is important so I don’t break it in a future version unintentionally.
  • Commit, do not mess with rakefile, version, or history. (if you want to have your own version, that is fine but bump version in a commit by itself I can ignore when I pull)
  • Send me a pull request. Bonus points for topic branches.
