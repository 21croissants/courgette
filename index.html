courgette by 21croissants

Courgette is a Rails engine (plugin) for Rails 2.x applications which makes your features files viewable through your browser

Dependencies

There are no dependencies to install to use the plugin. If courgette assets are missing - it should not happen as install script takes care of it. However, if something went wrong, you could do a manual setup by creating a symlink to courgette/public/courgette inside your application's public directory : $ cd public $ ln -s ../vendor/plugins/courgette/public/courgette courgette To contribute: * fork the project * bundle install
